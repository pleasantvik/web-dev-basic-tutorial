An Introduction to HTML

What is HTML?

HyperText Markup Language is a markup language. Use for structuring content on the web page. It is made of element, which contains the opening tag, closing tag and the content

What is Nesting?

Nesting is having an element within another element. Nesting follows the LIFO prinicple, which means Last in first out

What are void Elements

This are elements with a single tag. They do not follow the opening closing and content pattern. example of such elemnt is the image element

What is Attributes

This contains extra infomatioN about the element on which they are attached.
